🐍 Snake Game
A modern neon-themed Snake Game built with HTML, CSS, and JavaScript. The game combines smooth mechanics with a glowing cyberpunk-inspired interface, offering a fun and visually engaging experience across desktop and mobile devices.

✨ Features
🎮 Snake movement controlled via arrow keys

🍎 Food generation system with random placement

📊 Score counter that updates dynamically

💀 Game over detection when colliding with self

🔄 Restart / Play again functionality

🌌 Neon dark theme UI with glowing effects

📱 Responsive layout for desktop and mobile

🎨 Smooth animations and cyberpunk-inspired visuals

🛠️ Technologies Used
HTML5 – Canvas rendering

CSS3 – Neon theme and responsive design

JavaScript (ES6) – Game logic and interactivity

💻 Developed Using
Antigravity AI Assistant

Visual Studio Code / Antigravity IDE

🎯 Objective
The goal of the game is simple yet addictive:

Control the snake using arrow keys

Collect food to grow longer and increase your score

Avoid colliding with yourself

Challenge yourself to achieve the highest score possible
